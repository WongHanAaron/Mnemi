# Infrastructure

This directory contains infrastructure-as-code and deployment configurations for the Mnemi flashcard application.

## Contents

- **Dockerfiles**: Container images for web and API services
- **Kubernetes**: K8s manifests for orchestration (future)
- **CI/CD**: GitHub Actions workflows and scripts
- **MAUI**: Scripts and configurations for MAUI app distribution
- **Database**: Schema and migration scripts (future)

## Deployment Targets

- **Web**: Azure App Service or Docker containers
- **MAUI**: App Store / Google Play distribution pipelines
- **Database**: SQL Server or cloud-hosted alternatives
