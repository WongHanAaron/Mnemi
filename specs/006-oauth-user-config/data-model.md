# Data Model: OAuth User Configuration and Document Source Management

**Date**: 2026-05-02  
**Feature**: [spec.md](spec.md) | [plan.md](plan.md)

## Overview

This document defines all entities, value objects, and data structures for user authentication, OAuth connections, and document source configuration.

---

## Enumerations

### OAuthProvider

Identifies the external OAuth provider.

```csharp
public enum OAuthProvider
{
    /// <summary>Google OAuth provider for Google Drive access</summary>
    Google = 1,
    
    /// <summary>GitHub OAuth provider for repository access</summary>
    GitHub = 2
}
```

**Notes**:
- Values are spaced for future provider additions
- Used in AuthConnection and DocumentSource entities

### DocumentSourceProvider

Identifies the storage provider for a document source.

```csharp
public enum DocumentSourceProvider
{
    /// <summary>Google Drive folder</summary>
    GoogleDrive = 1,
    
    /// <summary>GitHub repository</summary>
    GitHub = 2
}
```

**Notes**:
- Mirrors OAuthProvider but semantically distinct (could support non-OAuth sources in future)
- Used in DocumentSourceConfig entity

## Core Domain Entities

### User

Aggregate root representing a person using Mnemi.

```csharp
public class User : Entity
{
    /// <summary>
    /// Unique identifier for the user.
    /// Generated by the system on creation.
    /// </summary>
    public Guid Id { get; private set; }
    
    /// <summary>
    /// User's email address (from OAuth provider).
    /// Used for account matching across providers.
    /// </summary>
    public string Email { get; private set; }
    
    /// <summary>
    /// Display name shown in the UI.
    /// Defaults to OAuth provider name but can be customized.
    /// </summary>
    public string DisplayName { get; private set; }
    
    /// <summary>
    /// UTC timestamp when the user account was created.
    /// </summary>
    public DateTime CreatedAt { get; private set; }
    
    /// <summary>
    /// Navigation property for OAuth connections.
    /// A user can have multiple connections (Google + GitHub).
    /// </summary>
    public ICollection<AuthConnection> AuthConnections { get; private set; }
    
    /// <summary>
    /// Navigation property for document sources.
    /// A user can have multiple sources across providers.
    /// </summary>
    public ICollection<DocumentSourceConfig> DocumentSources { get; private set; }
    
    // Factory method for creation
    public static User Create(string email, string displayName);
    
    // Domain methods
    public void UpdateDisplayName(string newDisplayName);
    public bool CanRemoveConnection(AuthConnection connection);
}
```

**Validation Rules**:
- `Email` must be a valid email format; required
- `DisplayName` must be 1-100 characters; required
- `Email` must be unique across all users

**Constraints**:
- User cannot be deleted if they have active document sources (must unlink first)
- At least one AuthConnection must remain (cannot orphan user)

### AuthConnection

Entity representing an OAuth connection between a Mnemi user and an external provider.

```csharp
public class AuthConnection : Entity
{
    /// <summary>
    /// Unique identifier for the connection.
    /// </summary>
    public Guid Id { get; private set; }
    
    /// <summary>
    /// Foreign key to the User entity.
    /// </summary>
    public Guid UserId { get; private set; }
    
    /// <summary>
    /// Navigation property to the owning user.
    /// </summary>
    public User User { get; private set; }
    
    /// <summary>
    /// The OAuth provider (Google or GitHub).
    /// </summary>
    public OAuthProvider Provider { get; private set; }
    
    /// <summary>
    /// Provider-specific user identifier.
    /// Google: 'sub' claim
    /// GitHub: 'id' claim
    /// </summary>
    public string ProviderUserId { get; private set; }
    
    /// <summary>
    /// Encrypted OAuth access token.
    /// Encrypted at rest using AES-256.
    /// </summary>
    public string EncryptedAccessToken { get; private set; }
    
    /// <summary>
    /// Encrypted OAuth refresh token (if provided).
    /// Encrypted at rest using AES-256.
    /// </summary>
    public string? EncryptedRefreshToken { get; private set; }
    
    /// <summary>
    /// UTC timestamp when the access token expires.
    /// Null if token doesn't expire.
    /// </summary>
    public DateTime? ExpiresAt { get; private set; }
    
    /// <summary>
    /// Space-separated list of granted OAuth scopes.
    /// </summary>
    public string Scopes { get; private set; }
    
    /// <summary>
    /// UTC timestamp when the connection was created.
    /// </summary>
    public DateTime CreatedAt { get; private set; }
    
    /// <summary>
    /// UTC timestamp of last update.
    /// </summary>
    public DateTime UpdatedAt { get; private set; }
    
    /// <summary>
    /// Indicates if the connection is valid and usable.
    /// Set to false if token refresh fails or user revokes access.
    /// </summary>
    public bool IsValid { get; private set; }
    
    // Factory method
    public static AuthConnection Create(
        Guid userId,
        OAuthProvider provider,
        string providerUserId,
        string accessToken,
        string? refreshToken,
        DateTime? expiresAt,
        string scopes);
    
    // Domain methods
    public void UpdateTokens(string accessToken, string? refreshToken, DateTime? expiresAt);
    public void MarkInvalid();
    public bool IsTokenExpired();
    public bool HasScope(string scope);
}
```

**Validation Rules**:
- `ProviderUserId` must not be null or empty; max 255 characters
- `EncryptedAccessToken` must not be null or empty
- `Scopes` must not be null or empty
- Combination of `Provider` + `ProviderUserId` must be unique (prevent duplicate OAuth accounts)

**Constraints**:
- Tokens are encrypted before storage; never stored in plain text
- `IsValid` is set to false when refresh fails; user must re-authenticate

### DocumentSourceConfig

Entity representing a linked external location containing flashcard content.

```csharp
public class DocumentSourceConfig : Entity
{
    /// <summary>
    /// Unique identifier for the document source.
    /// </summary>
    public Guid Id { get; private set; }
    
    /// <summary>
    /// Foreign key to the User entity.
    /// </summary>
    public Guid UserId { get; private set; }
    
    /// <summary>
    /// Navigation property to the owning user.
    /// </summary>
    public User User { get; private set; }
    
    /// <summary>
    /// Foreign key to the AuthConnection used to access this source.
    /// </summary>
    public Guid AuthConnectionId { get; private set; }
    
    /// <summary>
    /// Navigation property to the associated auth connection.
    /// </summary>
    public AuthConnection AuthConnection { get; private set; }
    
    /// <summary>
    /// The storage provider type.
    /// </summary>
    public DocumentSourceProvider Provider { get; private set; }
    
    /// <summary>
    /// User-defined display name for this source.
    /// Defaults to folder/repo name but can be customized.
    /// </summary>
    public string DisplayName { get; private set; }
    
    /// <summary>
    /// Provider-specific configuration as JSON.
    /// Google Drive: { "folderId": "..." }
    /// GitHub: { "repoOwner": "...", "repoName": "...", "rootPath": "..." }
    /// </summary>
    public string ProviderConfigJson { get; private set; }
    
    /// <summary>
    /// UTC timestamp when the source was created.
    /// </summary>
    public DateTime CreatedAt { get; private set; }
    
    /// <summary>
    /// UTC timestamp of last update.
    /// </summary>
    public DateTime UpdatedAt { get; private set; }
    
    /// <summary>
    /// Indicates if the source is currently accessible.
    /// Set to false if folder/repo is deleted or permissions revoked.
    /// </summary>
    public bool IsAccessible { get; private set; }
    
    /// <summary>
    /// Error message if source is not accessible.
    /// </summary>
    public string? LastErrorMessage { get; private set; }
    
    // Factory methods
    public static DocumentSourceConfig CreateGoogleDrive(
        Guid userId,
        Guid authConnectionId,
        string displayName,
        string folderId);
    
    public static DocumentSourceConfig CreateGitHub(
        Guid userId,
        Guid authConnectionId,
        string displayName,
        string repoOwner,
        string repoName,
        string? rootPath);
    
    // Domain methods
    public void UpdateDisplayName(string newDisplayName);
    public void MarkInaccessible(string? errorMessage);
    public void MarkAccessible();
    public T GetProviderConfig<T>() where T : class;
}
```

**Validation Rules**:
- `DisplayName` must be 1-200 characters; required
- `ProviderConfigJson` must be valid JSON; required
- Combination of `UserId` + `Provider` + provider-specific unique key must be unique
  - Google Drive: folderId must be unique per user
  - GitHub: repoOwner + repoName + rootPath must be unique per user

**Constraints**:
- Deleting an AuthConnection must cascade delete associated DocumentSourceConfigs
- `IsAccessible` is updated by background health checks or on access failure

## Value Objects

### GoogleDriveProviderConfig

Configuration specific to Google Drive document sources.

```csharp
public class GoogleDriveProviderConfig
{
    /// <summary>
    /// Google Drive folder ID.
    /// Found in the URL: drive.google.com/drive/folders/{folderId}
    /// </summary>
    public string FolderId { get; set; }
    
    /// <summary>
    /// Optional path within the folder for nested organization.
    /// </summary>
    public string? SubPath { get; set; }
}
```

**Validation**:
- `FolderId` must not be null or empty; typically 33-44 characters

---

### GitHubProviderConfig

Configuration specific to GitHub repository document sources.

```csharp
public class GitHubProviderConfig
{
    /// <summary>
    /// Repository owner (user or organization name).
    /// </summary>
    public string RepoOwner { get; set; }
    
    /// <summary>
    /// Repository name.
    /// </summary>
    public string RepoName { get; set; }
    
    /// <summary>
    /// Optional root path within the repository.
    /// Defaults to "/" for repository root.
    /// Example: "/notes/flashcards"
    /// </summary>
    public string RootPath { get; set; } = "/";
    
    /// <summary>
    /// Optional branch name.
    /// Defaults to "main" or "master".
    /// </summary>
    public string? Branch { get; set; }
}
```

**Validation**:
- `RepoOwner` must not be null or empty; max 39 characters (GitHub limit)
- `RepoName` must not be null or empty; max 100 characters (GitHub limit)
- `RootPath` must start with "/"; max 1000 characters

## Entity Relationships

```
┌─────────────┐       ┌──────────────────┐       ┌─────────────────────┐
│    User     │◄──────┤  AuthConnection  │◄──────┤ DocumentSourceConfig│
├─────────────┤  1:M  ├──────────────────┤  1:M  ├─────────────────────┤
│ Id (PK)     │       │ Id (PK)          │       │ Id (PK)             │
│ Email       │       │ UserId (FK)      │       │ UserId (FK)         │
│ DisplayName │       │ Provider         │       │ AuthConnectionId(FK)│
│ CreatedAt   │       │ ProviderUserId   │       │ Provider            │
└─────────────┘       │ EncryptedToken   │       │ DisplayName         │
                      │ Scopes           │       │ ProviderConfigJson  │
                      │ IsValid          │       │ IsAccessible        │
                      └──────────────────┘       └─────────────────────┘
```

**Relationship Rules**:
- User 1:M AuthConnection (cascade delete)
- User 1:M DocumentSourceConfig (cascade delete)
- AuthConnection 1:M DocumentSourceConfig (cascade delete)

## Database Schema (SQLite)

### Users Table

```sql
CREATE TABLE Users (
    Id TEXT PRIMARY KEY,  -- GUID as string
    Email TEXT NOT NULL,
    DisplayName TEXT NOT NULL,
    CreatedAt TEXT NOT NULL,  -- ISO 8601
    UNIQUE(Email)
);

CREATE INDEX IX_Users_Email ON Users(Email);
```

### AuthConnections Table

```sql
CREATE TABLE AuthConnections (
    Id TEXT PRIMARY KEY,
    UserId TEXT NOT NULL,
    Provider INTEGER NOT NULL,  -- 1=Google, 2=GitHub
    ProviderUserId TEXT NOT NULL,
    EncryptedAccessToken TEXT NOT NULL,
    EncryptedRefreshToken TEXT,
    ExpiresAt TEXT,  -- ISO 8601 or NULL
    Scopes TEXT NOT NULL,
    CreatedAt TEXT NOT NULL,
    UpdatedAt TEXT NOT NULL,
    IsValid INTEGER NOT NULL DEFAULT 1,  -- 0=false, 1=true
    FOREIGN KEY (UserId) REFERENCES Users(Id) ON DELETE CASCADE,
    UNIQUE(Provider, ProviderUserId)
);

CREATE INDEX IX_AuthConnections_UserId ON AuthConnections(UserId);
CREATE INDEX IX_AuthConnections_ProviderUserId ON AuthConnections(ProviderUserId);
```

### DocumentSources Table

```sql
CREATE TABLE DocumentSources (
    Id TEXT PRIMARY KEY,
    UserId TEXT NOT NULL,
    AuthConnectionId TEXT NOT NULL,
    Provider INTEGER NOT NULL,  -- 1=GoogleDrive, 2=GitHub
    DisplayName TEXT NOT NULL,
    ProviderConfigJson TEXT NOT NULL,
    CreatedAt TEXT NOT NULL,
    UpdatedAt TEXT NOT NULL,
    IsAccessible INTEGER NOT NULL DEFAULT 1,  -- 0=false, 1=true
    LastErrorMessage TEXT,
    FOREIGN KEY (UserId) REFERENCES Users(Id) ON DELETE CASCADE,
    FOREIGN KEY (AuthConnectionId) REFERENCES AuthConnections(Id) ON DELETE CASCADE
);

CREATE INDEX IX_DocumentSources_UserId ON DocumentSources(UserId);
CREATE INDEX IX_DocumentSources_AuthConnectionId ON DocumentSources(AuthConnectionId);
```
