# Home Dashboard Overview

## What is the Home Dashboard?

The Home Dashboard is your starting page for study mode. It gives a quick summary of where you are and what to study next.

## What You Can See

The page includes:
- A welcome area with a primary study action.
- Quick stats for motivation and daily progress.
- Recent decks you studied.
- Pinned decks you want fast access to.

## Deck Card Layout

Each deck card shows:
- A visual cover token.
- Deck title and subtitle.
- Status and progress summary.
- A primary button such as Open or Resume.

## Behavior in Different Data States

The dashboard handles common states clearly:
- Loading: sections show loading messages while data is fetched.
- Empty: sections show no-data messages when lists are empty.
- Missing data: fallback text appears when metadata is incomplete.

## Navigation

You can use the top navigation and side navigation to move between Home, Review, and Decks.

## Notes

This version focuses on desktop-first layout and is built from shared UI so it can be reused by both web and MAUI hosts.
