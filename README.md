# Mnemi Flashcard Web + MAUI Blazor App

A simple web and future MAUI Blazor app for creating flashcards from markdown text content and then running through those flashcards.

Planned features:
- Convert markdown into flashcards
- Support flashcards and definition matching game modes
- Add more study modes over time
- Future mode: guided question sequences that ask a user a predefined series of questions in order, leading them to the answer when they answer previous questions correctly
